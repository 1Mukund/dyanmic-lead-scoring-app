# Dynamic Lead Scoring App (Self-Learning)
1. Upload Excel file with behavioral lead data
2. App auto-learns feature importance (correlation-based)
3. Calculates score, bucket, suggests content and message
4. Shows logic behind scoring + weight breakdown
5. Run locally:
    pip install -r requirements.txt
    streamlit run lead_scoring_dashboard.py
6. Or deploy to Streamlit Cloud via GitHub
